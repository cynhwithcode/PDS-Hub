## 2026-06-02 컴포넌트 카드 클릭 방식 변경

### 배경
기존에는 카드 하단 "상세보기" 버튼으로만 이동 가능했음

### 고려한 옵션
- 옵션 A: 버튼 유지
- 옵션 B: 카드 전체 클릭 (eBay Playbook 레퍼런스)

### 결정
카드 전체 클릭으로 변경. Figma 링크 버튼만 별도 유지.

### 이유
카드가 하나의 오브젝트로 인식되어야 하고
버튼이 중복으로 존재할 필요 없음
